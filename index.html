Javier develops software for the Web and Android OS, although currently his main focus is learning about AI applications in Natural Language Processing.

The programming languages that he is most proficient at are Python, C++, Java, JavaScript, CSS and HTML.

A Progressive Webb App (PWA) that helps people practice the civics part of the citizenship test. It was created using Flask, Postgres DB and Service Workers(to cache content and make loading faster).

A Module for the open source medical record system OpenMRS. This module allows medical offices to keep digital copies of periodical surveys that help them monitor the recovery progress of their patients. It was created with the Spring Framework as a mixture of Java Server Pages (.jsp) and Groovy Server Pages (.gsp) using Java, Groovy, HTML, JavaScript, CSS, MySQL and Liquidbase.
